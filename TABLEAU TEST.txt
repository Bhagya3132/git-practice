Part A: MCQ Questions (10)
1. Which of the following is a Measure in Tableau?
A. Customer Name
B. Region
C. Sales
D. Category
Answer: C. Sales

2. What color represents Continuous fields in Tableau?
A. Blue
B. Red
C. Green
D. Yellow
Answer: C. Green

3. Which join returns only matching records from both tables?
A. Left Join
B. Right Join
C. Full Join
D. Inner Join
Answer: D. Inner Join

4. Which Tableau function returns the length of a string?
A. MID()
B. LEN()
C. LEFT()
D. COUNT()
Answer: B. LEN()

5. What is the output of ABS(-25)?
A. -25
B. 0
C. 25
D. Error
Answer: C. 25

6. Which function is used to replace NULL values with 0?
A. IFNULL()
B. ISNULL()
C. ZN()
D. COUNT()
Answer: C. ZN()

7. Which LOD expression does not react to most filters?
A. INCLUDE
B. EXCLUDE
C. FIXED
D. WINDOW_SUM
Answer: C. FIXED

8. Which filter is applied before data is loaded into Tableau?
A. Context Filter
B. Extract Filter
C. Measure Filter
D. Dimension Filter
Answer: B. Extract Filter

9. Which parameter type would you use for a Top N analysis?
A. String
B. Date
C. Integer
D. Boolean
Answer: C. Integer

10. Which table calculation provides ranking without skipping numbers?
A. RANK()
B. RANK_PERCENTILE()
C. INDEX()
D. RANK_DENSE()
Answer: D. RANK_DENSE()
